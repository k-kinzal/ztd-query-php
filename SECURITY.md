# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it through [GitHub's private vulnerability reporting](/security/advisories/new).

Please do **not** open a public issue for security vulnerabilities.

## Scope

This library is intended for use in test environments only. It is not designed to handle untrusted input in production systems. Security reports related to production use cases may be considered out of scope.
